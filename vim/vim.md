# Vim tutorial -- 
#### (IN Windows)
- 1. OPen powershell , if u have a file like :(fuck.txt), --  then u write cmd = vim fuck.txt,
- 2. then ur file will open in normal mode, {press i & (shift + a) to go to the normal mode to insert mode , then u able to write } 
- 3. if u want to quit press = :q
- 4. if u want to save press = :w
- 5. if u want to go to normal mode press = Esc
- 6. if u want to don't save = :!q
